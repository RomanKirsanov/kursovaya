Интерактивная демонстрация вероятностных алгоритмов в реальном времени.

## Запуск

 1. Установите зависимости
pip install -r requirements.txt

 2. Запустите приложение
python app.py

 3. Откройте в браузере
 http://localhost:5000

## Тестирование

pytest tests/

## Алгоритмы

Bloom Filter - проверка принадлежности с малым использованием памяти

HyperLogLog - оценка количества уникальных элементов

Count-Min Sketch - оценка частоты элементов в потоке
