# Интерактивная демонстрация вероятностных алгоритмов в реальном времени.

Возможности
 Реальная симуляция социальной сети

 Интерактивные графики в реальном времени

 Тестирование всех трех алгоритмов

 Отслеживание популярных хештегов

 Мониторинг использования памяти

## Технические требования для запуска проекта: 

Python 3.8+ 

Windows 10+ (64-bit) 

Минимальная версия: pip 21.0+ 

flask==3.0.0 

numpy==1.24.4 

mmh3==4.0.1 

Git 2.30+ 

Microsoft Visual C++ Build Tools (для mmh3) 

Требования к браузеру: 

Поддержка JavaScript ES6+ 

Поддержка WebSocket 

Включен JavaScript 

## Запуск

 1. Установите зависимости
pip install -r requirements.txt

 2. Запустите приложение
python app.py

 3. Откройте в браузере
 http://localhost:5000

## Тестирование

pytest tests/

## Алгоритмы

Bloom Filter - проверка принадлежности с малым использованием памяти

HyperLogLog - оценка количества уникальных элементов

Count-Min Sketch - оценка частоты элементов в потоке
