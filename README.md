# Интерактивная демонстрация вероятностных алгоритмов в реальном времени.

Возможности
 Реальная симуляция социальной сети

 Интерактивные графики в реальном времени

 Тестирование всех трех алгоритмов

 Отслеживание популярных хештегов

 Мониторинг использования памяти

## Запуск

 1. Установите зависимости
pip install -r requirements.txt

 2. Запустите приложение
python app.py

 3. Откройте в браузере
 http://localhost:5000

## Тестирование

pytest tests/

## Алгоритмы

Bloom Filter - проверка принадлежности с малым использованием памяти

HyperLogLog - оценка количества уникальных элементов

Count-Min Sketch - оценка частоты элементов в потоке
